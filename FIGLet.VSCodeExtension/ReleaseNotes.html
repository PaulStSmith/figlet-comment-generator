FIGlet Comment Generator - Release Notes

Version 1.2.3 - Update Release (April 2026)

Enhancements and Refactoring

  • VS Code extension now supports live preview of banners before insertion.
  • Improved font selection UI and added more bundled fonts.
  • Comment wrapping logic improved for more languages.
  • Performance improvements for large banners and files.
  • Updated to use the latest FIGLet TS Library for more accurate rendering.

Bug Fixes

  • Fixed an issue where banners were not inserted at the correct cursor position in some editors.
  • Resolved font preview glitches on some platforms.
