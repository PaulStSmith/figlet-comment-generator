Getting Started with FIGlet Comment Generator

The FIGLet Comment Generator extension brings the power of FIGLet ASCII art directly into Visual Studio Code. Create eye‑catching section headers, file banners, and comment blocks with just a few keystrokes — all rendered using the ByteForge FIGLet Suite engine.

🚀 Quick Start

The fastest way to create an ASCII art banner in VS Code:

  1. Place your cursor at the line where you want the banner
  2. Press Ctrl+Alt+B (Windows / Linux / macOS)
  3. Type your text
  4. Select a font (optional)
  5. Press Enter

Alternatively, open the Command Palette (Shift+Ctrl+P) and search for Generate FIGLet Banner.

✨ Features

  • Generate ASCII art banners using FIGLet fonts
  • Choose from multiple FIGLet fonts
  • Automatically wraps banners in the correct comment syntax
  • Supports multiple layout modes (Full Size, Kerning, Smushing)
  • Context‑aware insertion based on cursor position
  • Insert Class Banner — detects the class, interface, or struct at the cursor and pre‑fills the panel with its name
  • Insert Method Banner — same smart workflow for functions and methods
  • Smart insertion point: banners land above any existing doc‑comment block, not between it and the declaration
  • Customizable keyboard shortcuts
  • Retro‑terminal aesthetic with clean, modern UI

📖 Example

Input: Hello, World!

Output (using the “small” font, in a Python file):

#  | || |___| | |___      \ \    / /__ _ _| |__| | |
#  | __ / -_) | / _ \_     \ \/\/ / _ \ '_| / _` |_|
#  |_||_\___|_|_\___( )     \_/\_/\___/_| |_\__,_(_)
#                   |/

⚙️ Configuration

Open Settings using one of these methods:

  1. Press Shift+Ctrl+P and search for FIGLet Settings
  2. Right‑click any text editor and select FIGLet Comments > FIGLet Settings
  3. Press Ctrl+, (or go to File > Preferences > Settings) and search for FIGlet

You can configure:

  • Default font
  • Default layout mode
  • Font folder location

🧩 Supported Languages

The extension automatically detects the file type and uses the correct comment syntax:

  • C‑style languages (C, C++, C#, Java, JS, TS): // or /* */
  • Python, PowerShell, Ruby, Shell: #
  • HTML / XML: <!-- -->
  • SQL: --
  • Many more (20+ languages supported)

🔧 Powered By

This extension uses our own FIGLet TypeScript Library — part of the ByteForge FIGLet Suite — for rendering FIGLet text with full support for:

  • FLF font parsing
  • Layout modes (FullSize, Kerning, Smushing)
  • Smushing rules (Equal, Underscore, Hierarchy, Opposite Pair, Big X, Hardblank)
