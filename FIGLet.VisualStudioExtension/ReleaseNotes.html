FIGlet Comment Generator - Release Notes

Version 1.0.0 - Initial Release (November 2024)

🎉 Core Features

Context Menu Integration

  • Insert ASCII art banners through the context menu
  • Automatic class and method name detection
  • Smart indentation preservation

🔤 Language Support

Comprehensive language support with automatic comment formatting:

  • C-style Languages (C#, Java, C++, JavaScript)
    • Block comments (/* */)
    • Line comments (//)
  • Web Technologies
    • HTML/XML (<!-- -->)
    • CSS (/* */)
  • Scripting Languages
    • Python (# and docstrings)
    • PowerShell (# and <# #>)
    • Bash (#)
  • Database
    • SQL (-- and /* */)

🎨 Visual Studio Integration

  • Full Visual Studio theme support
    • Adapts to light and dark themes
    • Matches VS color scheme
  • Editor Integration
    • Preserves code formatting
    • Respects editor settings

⚙️ Configuration & Settings

  • Font Management
    • Custom font directory support
    • Live preview functionality
    • Multiple layout modes
  • Customization Options
    • Default layout preferences
    • Font selection memory

System Requirements

  • Visual Studio 2022 (17.0 or higher)
  • Windows (x64 platform)
  • .NET Framework 4.7.2 or higher

🔄 Known Limitations

  • Currently supports Visual Studio 2022 only
  • Windows x64 platforms only

🚀 Future Plans

  • Additional font repositories
  • Custom font creation tools
  • Multi-line text support
  • Banner style templates
  • Command palette integration
