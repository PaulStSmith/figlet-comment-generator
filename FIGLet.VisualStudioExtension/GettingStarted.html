Getting Started with FIGlet Comment Generator

The FIGLet Comment Generator extension brings FIGLet ASCII art directly into Visual Studio. Create bold, readable section headers and file banners inside your code with a single command — powered by the FIGLet .NET Library, the same engine used across the ByteForge FIGLet Suite.

🚀 Quick Start

The fastest way to create an ASCII art banner:

  1. Open any source‑code editor
  2. Right‑click where you want the banner and select Insert FIGLet Banner
  3. Enter your text in the dialog
  4. Choose a font
  5. Press OK

🛠 Installation

  1. Open Extensions → Manage Extensions
  2. Search for FIGLet Comment Generator
  3. Click Download
  4. Restart Visual Studio to complete installation

Supported versions: Visual Studio 2022  |  Visual Studio 2026

Creating Your First Banner

Method 1: Using the Context Menu

Right‑click in your code editor to access three banner options:

  • Insert FIGLet Banner — Creates a generic banner with your text
  • Insert FIGLet Class Banner — Automatically detects and uses the current class name
  • Insert FIGLet Method Banner — Automatically detects and uses the current method name
Pro Tip: When using Class or Method banners, the extension automatically suggests the detected name but allows you to modify it.

Method 2: Using the Edit Menu

You can also create banners through the Edit menu:

  1. Go to Edit > Insert FIGLet Banner
  2. Enter your text in the dialog
  3. Customize the appearance using available options
  4. Click OK to insert

Customizing Your Banner

Font Selection

Choose from various FIGlet fonts to style your banner:

  • Use the dropdown in the banner dialog to preview different fonts
  • The preview updates in real‑time as you type
  • Your last used font is remembered for next time

Layout Options

Adjust the banner layout using these modes:

  • Full Width — Characters use maximum width
  • Fitted (Kerning) — Removes unnecessary spacing
  • Smushed — Combines characters where possible (default)

🧩 Supported Languages

The extension automatically detects the file type and applies the correct comment style:

  • C# / C++ / Java → // or /* */
  • F# → //
  • VB.NET → '
  • SQL → --
  • JavaScript / TypeScript → //
  • Many others supported through Visual Studio's language service

Configuration

Settings

Go to Tools > Options > FIGLet Comment Generator to configure:

  • Default FIGLet font
  • Default layout mode
  • FIGLet font folder

Adding Custom Fonts

  1. Go to Tools > Options > FIGLet Comment Generator
  2. Set your custom font directory
  3. Add .flf files to this directory
  4. Fonts will appear in the selection dropdown

Best Practices

  • Use banners for major code sections to improve navigation
  • Choose fonts that balance readability and style
  • Maintain consistent banner styles across your project
  • Consider your team's coding standards when using ASCII art comments
